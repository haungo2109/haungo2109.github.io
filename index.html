Cách cài đặt

B1: check mail click vào View invitation rồi accept invitation

B2: tải git theo link https://git-scm.com/downloads rồi cài đặt. (khi cài next hết)

B3: tải tortoisegit theo link https://tortoisegit.org/download/ rồi cài đặt bấm next hết đến khi yêu cầu điền tên đăng nhập vs email là xong

B4: tạo 1 thư mục trên máy để lưu bài.

B5: vào thư mục đã tạo click chuột phải vào vùng trống chọn Git Clone

B6: copy link nhóm vào URL rồi chọn OK

Cách nộp bài

B1: vào thư mục lưu bài click chuột phải chọn TortoiseGit-->Pull

B2: copy file bài tập đã làm trước đó vào thư mục tương ứng trên máy nếu chưa có thì tạo ra như hình dưới

B3: ra thư mục lưu bài click chuột phải vào vùng trống chọn Git commit --> “master”.

B4: nhập nội dung thay đổi vào Message (gì cũng đc) và chọn những file muốn nộp lên Github --> commit&push. (sẽ hiện lên bảng đăng nhập tài khoản github đối với lần nộp bài đầu tiên cứ đăng nhập bình thường)

Cách xóa bài

B1: vào thư mục lưu bài click chuột phải vào vùng trống chọn TortoiseGit-->Pull

B2: xóa file muốn xóa trong thư mục lưu bài tập.

B3: click chuột phải vào vùng trống trong thư mục lưu bài tập chọn Git commit --> “master”.

B4: nhập nội dung thay đổi vào Message (gì cũng đc) và chọn những file đã xóa để cập nhật lên Github, rồi nhấn commit&push.
